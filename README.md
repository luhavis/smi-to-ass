# Subtitle convert (SMI to ASS)

# How to use
```bash
# Default
python main.py --smi_dir "./smi" --save_dir "./ass"

# Add styles
python main.py --smi_dir "./smi" --save_dir "./ass" --styles "1-Style" "2-Style" "3-Style"
```

### Create conda env
```bash
conda create -n smi-to-ass python=3.11
```

### Activate conda env
```bash
conda activate smi-to-ass
```
